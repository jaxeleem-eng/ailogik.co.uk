# Logik AI

Logik AI specializes in sourcing qualified, pipeline-ready leads for AI companies. This website provides an overview of our services, process, and success stories.

## Project Structure

- `index.html`: Main landing page
- `assets/`:
  - `styles.css`: Custom CSS for branding and layout
  - `app.js`: Interactive components and interactions
  - `logo.jpg`: Company logo
  - `banner.jpg`: Site banner

## Features

- Sleek, professional AI-focused design
- Testimonials from real clients
- Fully responsive for mobile and desktop
- Interactive contact form
- Multi-channel outreach focus

## Local Development

To run the site locally, you can use any static server. If you are on Windows, you can use the provided PowerShell script:

```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1
```

Or if you have Python:

```bash
python -m http.server 5500
```
